3D Babel

3D environment of the Library of Babel (https://libraryofbabel.info/) in Unity

Download .zip Download .tar.gz View on GitHub

Bienvenido

Con 3D Babel podrás probar una experiencia de inmersión 3D de la biblioteca de babel (https://libraryofbabel.info/) en un experimento para generación de paseos virtuales de páginas web.

3D Babel es libre, alojado en github bajo licencia GPLv3.
